# Task Manager

A simple Task Manager app built with React.

## Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed or pending
- Search tasks
- Filter tasks (All / Completed / Pending)
- Dark mode (Settings menu)
- Save tasks in localStorage

## Tech Stack
- React
- JavaScript
- HTML
- CSS


## How to Run
1. Install dependencies:

npm install

2. Run the app:
   
npm start
